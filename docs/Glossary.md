# Glossary

- Lead: A potential customer record with status workflow (new, working, qualified, converted, disqualified)
- Opportunity: Potential sale linked to a lead/client with stage pipeline
- Activity: Task/Note/Call logged against lead/client/etc. via related morph
- Client: Customer entity linked to a user
- Booking: Service booking for a client
- Ticket: Support case with SLA and status history
- Pipeline/Stage: CRM sales process structure
- Morph Map: Alias mapping for polymorphic relations
